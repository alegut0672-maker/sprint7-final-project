# sprint7-final-project
Analizar el comportamiento de los clientes de ConnectaTel para identificar segmentos clave, detectar patrones de uso y outliers, y proponer recomendaciones estratégicas para optimizar planes y mejorar la retención.
Datasets utilizados: Datos de clientes que incluyen variables de edad, cantidad de mensajes enviados, cantidad de llamadas realizadas y minutos totales de llamadas.

Etapas del análisis realizadas:
Diagnóstico inicial y detección de outliers.
Segmentación de clientes por uso y edad.
Identificación de segmentos estratégicos y patrones de uso extremo.
Formulación de recomendaciones comerciales y estratégicas.

Cómo ejecutar el codigo:

Abrir el notebook en Google Colab o en un entorno local con Jupyter.
Cargar los datasets proporcionados.
Ejecutar las celdas en orden para reproducir el análisis.

Guía breve de reproducción:

Verificar que las librerías necesarias estén instaladas (pandas, numpy, matplotlib, seaborn).
Cargar los datos y realizar limpieza básica.
Ejecutar análisis exploratorio para detectar outliers y segmentar.
Generar visualizaciones y tablas resumen.
Revisar recomendaciones y conclusiones para la toma de decisiones.
